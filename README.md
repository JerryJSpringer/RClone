# RClone
A simplified re-creation of Reddit in C# using ASP.NET Core MVC with Entity Framework Core.

# Features
### Implemented
  - User
    - Admin and owner privileges
    - Password change
    - Public username change
  - Communities
    - Basic moderator support
    - Subscriptions
    - Posting
  - Posts
    - Commenting
    - Voting systems
  - Explore system to find new communities
